# MSCschedulizer
## Installation
- ```bower install```
- ```npm install```
- Copy /asset/js/config.example.js to /asset/js/config.js
- ./node_modules/browserify/bin/cmd.js asset/js/bundleme.js -o asset/js/schedulizer_bundle.js
- ./node_modules/mocha/bin/mocha -r mock-local-storage
