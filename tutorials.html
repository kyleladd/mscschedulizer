Overview

The responsively-themed app gets real-time data and manages the latest four semesters. It replaces our current course listings pages. It enables students to select their classes, generate schedules, and pick the one that best suits them. To aid in the course selection process there are various filters that can be applied: Morrisville Campus, Norwich Campus, not full (current enrollment is less than max enrollment), online, or having time blocks (times when you don’t want schedule classes). Once a schedule is selected, it (the schedule’s CRNs) can be imported directly into the Web for Students class registration form. This latest version, written in HTML5, CSS3, and JavaScript and interfacing with the C# API using HTTP requests, allows for the app to be easily ported into Android applications, Windows Desktop applications, and a website with all of them using the same code using Electron and Cordova.

Selecting Classes

There are a few ways to select courses within the schedulizer. One way is to use the Select Classes page which will select the class with all of its sections to be used within the generated combinations. This functionality is the same as using the Course details page and clicking the plus icon next to a course's title. To remove a selection, click on the corresponding listing from within the selections section of the sidebar.

Favoriting Schedules

Favoriting schedules allows you to be able to compare schedules that have different courses in them. For instance, you could generate schedules with the following classes selected: CITA 120, CITA 140, CITA 460. On the generate page, a list of valid schedule combinations will be displayed. You could choose the schedules that you like most and favorite them. Then, you could change your course selections to the following: CITA 120, CITA 140, BSAD 116, BSAD 300. Once you generate the schedules with the new class combinations you can then favorite your preferred schedules. Then, to compare all of your favorite schedules (which may include different class selections) you can go to the favorites page which will list only your favorited schedules.

Exporting to Web For Students

Exporting your schedule into Web for Students is as easy as a couple of clicks. Below each schedule there is an export link. By clicking export, each section within the selected schedule will have its Course CRNs exported into a cookie that is accessible on Morrisville.edu. Then, once signed in to Web for Students, go to the intended semester's add/drop class form. Above the form there will be a button to import the Course CRNs from the schedulizer cookie and load them into the form for you. Click submit and you're all set.

  1. Select Classes
  2. Generate Schedules
  3. Click the export button below your perfect schedule.
  4. Log in to Web for Students
  5. Go to the Add/Drop form for the intended semester
  6. Click the button entitled "Import from MSCschedulizer"
  7. Click "Submit Changes" to sign up for those classes and complete scheduling

CRN Selections

Perhaps you want only a particular section or sections of a course to be in the result of the combinations of schedules. When multiple sections are selected that are of the same type (lectures, labs), they are treated as an "OR". However, if the crn selections are of different types, they are treated as an "AND".

Visual Filters

By using the visual filter, you're able to remove sections from consideration. Once your courses are selected (via the select classes or the course listings pages), you're able to view all of the sections of all of your selected courses onto one schedule via the Visual Filters page. By clicking on a particular section you're able to see the details for that section as well as the ability to remove the section from consideration. When a section is removed from consideration, it will not appear in any of the generated combinations on the Generate page.

Using Visual Filters

To add a visual filter, (after selecting your courses) go to the visual filters page (accessible via the main navigation). To view your current schedule filters and remove a visual filter, above the schedule section on the Generate and Visual Filters pages, click the visual filters button. This will then list out all visual filters. Alongside each filter will be the ability to delete the filter (thus adding the section back into consideration).
